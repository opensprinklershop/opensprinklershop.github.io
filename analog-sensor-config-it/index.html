Configurazione sensori analogici

Analog Sensor Configuration è l’area UI OpenSprinklerPro per definire sorgenti sensore e usare i valori nell’automazione dell’irrigazione. Si apre dal menu inferiore con Analog Sensor Configuration.

Pagine correlate: Automazione sensori e monitor, Sensori FYTA, Estensione OpenSprinklerPro, addendum API e piattaforme.

Definizioni sensori

La prima sezione definisce la lista sensori. Ogni riga contiene numero, nome, valore corrente e stato attivo. Add Sensor apre l’editor di un nuovo sensore; i sensori esistenti si modificano toccando il nome.

Famiglie supportate secondo piattaforma e firmware:

Famiglia Uso tipico
Scheda analogica Valori tensione o percentuale da sensori analogici
Truebner / RS485 / Modbus SMT50/SMT100/TH100 e valori Modbus RTU generici
Zigbee Sensori ESP32-C5 Zigbee, ad esempio temperatura, umidità o suolo
BLE Sensori Bluetooth Low Energy su build ESP32 con BLE
MQTT Valori pubblicati da sistemi esterni
FYTA Umidità e temperatura del suolo dal cloud FYTA
OpenSprinkler remoto Valore letto da un altro controller
Meteo Temperatura, umidità, pioggia, vento, ETo o radiazione
Gruppi Minimo, massimo, media o somma di altri sensori
Diagnostica Memoria libera, spazio libero o diagnostica controller

Regolazioni programma

Program Adjustments adatta la durata di irrigazione di un programma in base a un valore sensore.

Tipi: LINEAR, DIGITAL_MIN, DIGITAL_MAX, DIGITAL_MINMAX. Vedi Automazione sensori e monitor e addendum API.

Monitor e avvisi locali

Monitoring and Control crea regole di soglia e logica dai sensori. I monitor possono generare avvisi locali e alimentare le notifiche.

Grafici e dati sensore

Sensor Chart mostra valori registrati, scarica log o apre la tabella dati.

Configurazione FYTA

FYTA Setup salva token API o credenziali FYTA e mappa i sensori pianta in OpenSprinkler. Vedi Sensori FYTA.

Backup e ripristino

Backup and Restore esporta o ripristina configurazione sensori, regolazioni programma e monitor separatamente dal backup generale.
