OpenSprinkler Shop Firmware and Documentation

This site contains multilingual documentation for OpenSprinkler firmware and the OpenSprinklerPro extensions maintained by OpenSprinklerShop.

OpenSprinklerShop firmware version

The OpenSprinklerShop firmware, including OpenSprinklerPro, uses its own firmware version numbering independent of the upstream OpenSprinkler firmware line. The current OpenSprinklerShop firmware version is 2.4.0(199).

  • User manual: English, Deutsch, Français, Italiano.
  • OpenSprinklerPro extensions: English, Deutsch, Français, Italiano.
  • Firmware release notes: OpenSprinklerShop Firmware Releases.

Firmware-Update

OpenSprinklerPro Online-OTA

OpenSprinklerPro nutzt den UI-Dialog Seitenmenü → Online-Update. Der Dialog prüft das Manifest, erstellt ein Backup, wählt die passende Firmwarevariante und zeigt den Update-Fortschritt.

Details: OpenSprinklerPro Online-OTA.

OpenSprinkler v3

Klassische OpenSprinkler-3.x-Firmware wird über http://<controller-ip>/update aktualisiert. Vorher über das Seitenmenü Konfiguration exportieren.

OpenSprinkler Pi (OSPi)

OSPi wird direkt auf dem Raspberry Pi über Shell/Git/Build-Skripte aktualisiert, nicht über den Web-OTA-Dialog.

Hardware-Schnittstelle

OpenSprinklerPro hat eine eigene Hardware-Schnittstelle auf ESP32-C5-Basis. Details und Bild: Hardware-Schnittstelle im Benutzerhandbuch.

Firmware Update

OpenSprinklerPro Online OTA

OpenSprinklerPro uses Side menu → Online Update. The dialog checks the manifest, creates a backup, selects the correct firmware variant and displays update progress.

Details: OpenSprinklerPro Online OTA.

OpenSprinkler v3

Classic OpenSprinkler 3.x firmware is updated through http://<controller-ip>/update. Export the configuration from the side menu first.

OpenSprinkler Pi (OSPi)

OSPi is updated directly on the Raspberry Pi using shell/Git/build scripts, not through the firmware web OTA dialog.

Hardware Interface

OpenSprinklerPro has its own ESP32-C5-based hardware interface. Details and image: Hardware Interface in the user manual.
